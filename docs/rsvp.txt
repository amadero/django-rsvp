===============================
django-rsvp: A simple RSVP app.
===============================


Requirements
============

```django-rsvp``` requires Django-1.0_beta1 or better.


Quick Start
===========

#. Add ```rvsp``` app to your ```INSTALLED_APPS``.
#. Set the ```RSVP_FROM_EMAIL``` setting to an e-mail address you'd like invites to be sent from.
#. Run ```./manage.py syncdb```.
#. Add ```(r'^rsvp/', include('rsvpproject.rsvp.urls')),``` to your ```urls.py```.
